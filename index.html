Equação de 2º grau:
O que é?

A equação do segundo grau recebe esse nome porque é uma equação polinomial cujo termo de maior grau está elevado ao quadrado. Também chamada de equação quadrática, é representada por: ax² + bx + c = 0

Numa equação do 2º grau, o x é a incógnita e representa um valor desconhecido. Já as letras a, b e c são chamadas de coeficientes da equação. Os coeficientes são números reais e o coeficiente a tem que ser diferente de zero, pois do contrário passa a ser uma equação do 1º grau.

As equações do 2º grau completas são aquelas que apresentam todos os coeficientes, ou seja a, b e c são diferentes de zero (a, b, c ≠ 0). Por exemplo, a equação 5x2 + 2x + 2 = 0 é completa, pois todos os coeficientes são diferentes de zero (a = 5, b = 2 e c = 2).Já uma equação é incompleta quando pelo menos um dos seus coeficientes é zero.
